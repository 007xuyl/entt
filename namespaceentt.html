◆ ident

template<typename... Types>
constexpr auto entt::ident = Identifier<std::decay_t<Types>...>{std::make_index_sequence<sizeof...(Types)>{}}

Types identifers.

Variable template used to generate identifiers at compile-time for the given types. Use the constexpr get member function to know what's the identifier associated to the specific type.

Note
Identifiers are constant expression and can be used in any context where such an expression is required. As an example:
constexpr auto identifiers = entt::ident<AType, AnotherType>;
switch(aTypeIdentifier) {
case identifers.get<AType>():
// ...
break;
case identifers.get<AnotherType>():
// ...
break;
default:
// ...
}
Template Parameters
TypesThe list of types for which to generate identifiers.
